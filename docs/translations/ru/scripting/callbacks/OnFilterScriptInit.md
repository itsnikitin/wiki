## Описание

Этот callback вызывается, когда filterscript инициализируется (загружается). Он вызывается только внутри скрипта, который запускается.

## Примеры

```c
public OnFilterScriptInit()
{
    print("\n--------------------------------------");
    print("Ваш filterscript загружен.");
    print("--------------------------------------\n");
    return 1;
}
```
